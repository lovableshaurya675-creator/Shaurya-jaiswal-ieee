# Task 1 - UI Clone Challenge | IEEE MANIT Web Dev Recruitment

## What I did
Recreated the Crypko anime character generation webpage as a UI clone task.

## Pages / Files
- `index.html` - main page structure
- `style.css` - all styling
- `script.js` - small JS interactions
- `anime bottom.jpg` - left anime image
- `anime_right.jpg` - right anime character card image

## Features
- Exact layout clone of the given reference image
- Black rounded navbar with hover underline animation on links
- Large bold heading — ANIME CHARACTER GENERATION
- Rotating SVG circle badge with START GENERATE button
- Right anime girl card with yellow accent bar
- Bottom right info card with Download button
- Yellow CREATE ANIME button
- Hover effects on buttons
- Responsive for mobile screens
